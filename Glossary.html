HMTL

HTML stands for Hypertext Mark-up language and it’s used to create a web page. HTML has a number of various tags which allows the developer to make up their content with semantic so it helps paragraphs to known as text copy, while images are imported as images.

JavaScript

JavaScript is a scripting language that is used to create interactions on a web page. It helps to implement various complex things on web pages such as content updates, interactive maps, animated 2D/3D graphics, scrolling video jukeboxes, etc. Many developers use JavaScript because it has the ability to load new content without having to reload the whole page.

CSS

HTML isn't enough on its own to create beautiful web pages, however. Cascading Style Sheets (CSS) provide a method for designers to create a visual set of rules that determine how different elements within a web page will be rendered on screen by the browser.CSS allows things like the colour of text, the background, the size, shape and position of all the different parts of a page to be set.

A responsive design

A responsive design is one that adapts to the user’s device and, in an ideal world, the user’s context so that it helps to present the content essential in the most accessible way. In general, it tells us that a web page will re-paginated itself as the screen size reduces or increases, helps to present in multiple columns when its viewed on a desktop computer.

HTTPS

HTTPS stands for (Hyper Text Transfer protocol secure) is the secure version of HTTP, this protocol is used to send data between a web browser and a website. HTTPS uses a Secure Socket layer (SSL) certificate and it is used to encrypt all communication between your browser and the website.

API

An API stands for (application programming interface) is an instruction that allows various system to communicate. It delivers a technique of accessing or manipulating data from a system, typically delivered by the system. Developers create APIs so that data which is exits in their system can be used or made in various place.

Ajax

It represents Asynchronous JavaScript and XML. AJAX is ordinarily utilised for making dynamic web applications and allows for asynchronous information recover without reloading the page a client is on. The JavaScript is on given page handles the vast majority of the fundamental elements of the application, influencing it to perform increasingly like a desktop program rather than an online one.

Accessibility

Fundamentally, this is the capacity of a site to be utilised by individuals with disabilities, including outwardly weakened visitors utilised screen readers, hearing debilitated visitors utilising no sound, partially blind individuals, or those with different disabilities. A site with low accessibility is fundamentally going to be unimaginable for those with disabilities to utilise. Accessibility is especially significant for sites delivering information to those with handicaps (human services sites, government sites. However, it is essential perspective to think about when planning any site.
